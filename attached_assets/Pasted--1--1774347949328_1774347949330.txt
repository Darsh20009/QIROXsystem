1. الصفحة الرئيسية
الرابط: /

المميزات:

شريط إعلانات متحرك (Marquee) يعرض: شحن مجاني، كود خصم QIROX10، ضمان الجودة
بانر رئيسي (Carousel) من ٣ شرائح تتغير تلقائياً كل ٥ ثوانٍ مع تأثيرات Fade، ويمكن التنقل يدوياً
الشريحة ١: كولكشن ٢٠٢٥ — "أناقة لا حدود لها"
الشريحة ٢: عروض حصرية — "خصومات استثنائية"
الشريحة ٣: شحن مجاني — "توصيل سريع لباب بيتك"
قسم الفئات (٦ بطاقات): ملابس، أحذية، إكسسوار، عروض، الأكثر مبيعاً، وصل حديثاً
شريط الثقة الأسود (٤ عناصر): شحن مجاني / إرجاع مجاني / دفع آمن ١٠٠٪ / دعم ٢٤/٧
قسم عروض اليوم مع عداد تنازلي (ساعة: دقيقة: ثانية)
بانرات ترويجية مزدوجة جنباً لجنب (أحدث التشكيلات / تخفيضات حتى ٥٠٪)
قسم اختيارات العملاء (أفضل المنتجات)
شريط الإحصائيات: +١٠٠٠ عميل / +٢٠٠ منتج / ٩٩٪ رضا / ٢-٤ أيام توصيل
قسم Brand Manifesto الأسود مع الشعار وجميع وسائل الدفع
زر الواتساب العائم في أسفل الشاشة
Footer كامل: الشعار، التصنيفات، الدعم، وسائل التواصل (Instagram, X, Snapchat, TikTok, LinkedIn)
أيقونات وسائل الدفع: مدى، فيزا، ماستركارد، Apple Pay، STC Pay، تمارة، تابي، أمريكان إكسبريس
شهادة مركز الأعمال السعودي
2. صفحة المنتجات
الرابط: /products

المميزات:

عرض جميع المنتجات في شبكة (Grid) ٢-٤ أعمدة حسب الجهاز
بطاقة كل منتج تعرض: الصورة، الاسم، السعر، حالة المخزون
زر إضافة سريع للسلة من البطاقة مباشرة
فلترة وبحث عبر المنتجات (إذا مُفعَّل)
تأثيرات Hover على الصور
3. صفحة تفاصيل المنتج
الرابط: /products/:id

المميزات:

صور المنتج بعرض كامل
الاسم، الوصف، السعر
اختيار اللون والمقاس (إذا توفرت خيارات)
الكمية المتاحة في المخزون
زر إضافة للسلة مع تأثير بصري
المنتجات المشابهة (إن وُجدت)
4. سلة التسوق
الرابط: /cart

المميزات:

عرض جميع المنتجات المضافة مع الصور والأسعار
تعديل الكمية أو حذف منتج
حساب المجموع الفرعي + ضريبة ١٥٪ + الشحن
تطبيق كود الخصم
عرض رصيد المحفظة (إذا كان مسجلاً)
زر الانتقال لإتمام الطلب
5. صفحة إتمام الطلب (Checkout)
الرابط: /checkout

المميزات:

إدخال عنوان التوصيل الكامل
اختيار شركة الشحن
اختيار طريقة الدفع:
بطاقة بنكية → ينتقل لبوابة الدفع
STC Pay → ينتقل لبوابة الدفع
Apple Pay → ينتقل لبوابة الدفع
تمارة → ينتقل لصفحة تمارة
تابي → ينتقل لصفحة تابي
محفظة Qirox → يخصم مباشرة
تحويل بنكي → يطلب رفع صورة الإيصال
ملخص الطلب بالكامل قبل التأكيد
إرسال إشعار فوري + بريد إلكتروني عند إنشاء الطلب
6. بوابة الدفع
الرابط: /payment/gateway?orderId=...&amount=...

المميزات:

٣ تبويبات: بطاقة بنكية / STC Pay / Apple Pay
تبويب البطاقة:
بطاقة افتراضية متحركة تعكس ما تكتبه (flip animation)
كشف تلقائي لنوع البطاقة (مدى / فيزا / ماستركارد) بالأيقونة
التحقق من صحة البطاقة بخوارزمية Luhn في الوقت الفعلي
نموذج: رقم البطاقة، اسم الحامل، تاريخ الانتهاء، CVV
نافذة 3DS OTP (٦ أرقام) مع عداد تنازلي ٦٠ ثانية
تأثيرات loading واقعية (١.٨-٣ ثانية معالجة)
تبويب STC Pay:
إدخال رقم الجوال
إرسال OTP وهمي (OTP: 1234)
نافذة إدخال OTP
تبويب Apple Pay:
زر Apple Pay الأسود الرسمي
محاكاة Face ID بتأثيرات بصرية
نتيجة ناجح/فاشل واقعية

9. تسجيل الدخول
الرابط: /login

10. إنشاء حساب
الرابط: /register

المميزات:

حقول: الاسم الكامل، رقم الجوال، البريد الإلكتروني، كلمة المرور، تأكيد كلمة المرور
التحقق التلقائي من عدم تكرار الحساب
إرسال بريد ترحيب تلقائي عند التسجيل (إذا أدخل بريداً حقيقياً)
تسجيل دخول تلقائي بعد إنشاء الحساب

17. إدارة المنتجات

18. إدارة الطلبات
(داخل لوحة التحكم /admin)

تغيير حالة الطلب (جديد → قيد التجهيز → تم الشحن → مُسلَّم)
إدخال شركة الشحن ورقم التتبع (عند الشحن)
إلغاء الطلب مع السبب
عند كل تغيير حالة: إشعار فوري للعميل + بريد إلكتروني تلقائي

المميزات:

📱 مميزات إضافية شاملة
نظام البريد الإلكتروني (SMTP2GO)
يُرسَل تلقائياً في هذه الحالات:

الحدث	القالب	المحتوى
إنشاء طلب	تأكيد الطلب	جدول المنتجات، الإجمالي، العنوان، طريقة الدفع
تغيير الحالة لـ "قيد التجهيز"	تحديث الطلب	إشعار بدء التجهيز
تغيير الحالة لـ "تم الشحن"	تحديث الطلب	رقم التتبع وشركة الشحن
تغيير الحالة لـ "تم التسليم"	تحديث الطلب	رسالة شكر وتأكيد
إلغاء الطلب	تحديث الطلب	سبب الإلغاء
إتمام الدفع بالبطاقة	إيصال الدفع	رقم العملية، كود الموافقة، المبلغ
إتمام الدفع بـ STC Pay	إيصال الدفع	رقم العملية، المبلغ
إتمام الدفع بـ Apple Pay	إيصال الدفع	رقم العملية، المبلغ
التسجيل الجديد	ترحيب	مميزات المتجر، رابط التسوق
نظام الإشعارات (٣ طبقات)
الطبقة	التقنية	الوصف
داخل التطبيق	جرس الإشعارات	تظهر فوراً لجميع المستخدمين المتصلين
إشعارات المتصفح	Web Push (VAPID)	تصل حتى لو أُغلق المتصفح
قاعدة البيانات	MongoDB	محفوظة دائماً ويمكن مراجعتها لاحقاً
التطبيق PWA (Progressive Web App)
يمكن تحميله على الجوال (iPhone / Android) كتطبيق أصلي
يعمل بدون انترنت (للصفحات المحفوظة)
أيقونة على الشاشة الرئيسية
زر "تحميل التطبيق" يظهر في الـ Navbar تلقائياً
ثنائية اللغة (AR / EN)
زر تبديل اللغة في الـ Navbar (أيقونة الكرة الأرضية)
جميع النصوص تتغير فوراً (بدون إعادة تحميل)
اتجاه الصفحة يتغير تلقائياً (RTL ↔ LTR)
نظام الولاء والمحفظة
نقاط تُجمَع مع كل طلب
مستويات: برونز → فضي → ذهبي
المحفظة الرقمية: رصيد يمكن الدفع به مباشرة
عرض الرصيد في القائمة المنسدلة للحساب
