# تطوير تسليم المشروع - فيديو وملفات وطريقة استخدام

## What & Why
تطوير نظام تسليم المشروع للعميل بحيث يستطيع الموظف/المدير إرفاق فيديو شرح وملفات (PDF/مستندات) وكتابة طريقة استخدام تفصيلية عند تسليم المشروع. حالياً usageGuide يدعم فقط عنوان ووصف نصي وروابط ملفات خارجية (URLs يدوية) بدون رفع مباشر ولا عرض فيديو مدمج.

## Done looks like
- الموظف يستطيع رفع فيديو شرح مباشرة (بدون قيد حجم) عند تسليم المشروع
- الموظف يستطيع رفع ملفات إضافية (PDF، مستندات) مباشرة
- حقل طريقة الاستخدام يدعم نص طويل مع خطوات مرقمة
- في لوحة العميل (Dashboard): الفيديو يُعرض مباشرة بمشغل مدمج مع زر تحميل، الملفات تظهر كروابط تحميل/فتح
- صفحة تفاصيل المشروع (ProjectDetails) تعرض التسليمات بشكل واضح ومنظم

## Out of scope
- نماذج الأعمال في صفحة الخدمات (مهمة منفصلة)
- تغيير مراحل المشروع أو الإشعارات

## Tasks
1. **تحديث نموذج المشروع (Schema)** — توسيع usageGuide في projectSchema لإضافة: videos (مصفوفة كائنات url, name), attachments (مصفوفة كائنات url, name, type) بجانب الحقول الموجودة.
2. **تطوير واجهة AdminOrders** — في قسم تسليم المشروع: إضافة رفع فيديو مباشر، رفع ملفات مباشر (مع عرض الملفات المرفقة)، تحسين حقل طريقة الاستخدام.
3. **تطوير عرض العميل** — في Dashboard.tsx و ProjectDetails.tsx: عرض فيديو التسليم بمشغل مدمج + زر تحميل، عرض الملفات المرفقة كقائمة تحميل، عرض طريقة الاستخدام بتنسيق واضح.
4. **ربط مع endpoint الرفع الكبير** — استخدام endpoint رفع الملفات الكبيرة (من المهمة الأولى) لرفع الفيديوهات والملفات الكبيرة.
